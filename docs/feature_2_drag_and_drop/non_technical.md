# Product Overview: Drag & Drop Mapping

## What it does
The Drag & Drop interface is the core workspace of the application. Users can grab a specific type of data field (like a Signature box, Date, or Text input) from the left-hand menu and drop it exactly where they want it to appear on their uploaded PDF. Once placed, they can easily click and drag it again to fine-tune the position.

## User Benefits

* **WYSIWYG Experience:** What You See Is What You Get. The exact spot where the user releases their mouse cursor is precisely where the variable locks in. There is no guessing or manual coordinate typing required.

* **Responsive Accuracy:** Because of the math powering the layout, if a user builds a template on a massive 4k desktop monitor, and another user opens that same template on a small 13-inch laptop, the variable boxes will scale perfectly and stay glued to the exact same words on the PDF.

* **Visual Clarity:** Each variable type has its own distinct color and icon (e.g., Purple for Signatures, Green for Numbers). This allows users to understand the layout of a complex document at a single glance.

## Business Value
This visual approach drastically lowers the technical barrier to entry. Instead of requiring developers or data-entry specialists to manually code where text should go on a document, any standard business user (like HR or Sales) can visually map out a contract in seconds, saving immense amounts of time and operational cost.
