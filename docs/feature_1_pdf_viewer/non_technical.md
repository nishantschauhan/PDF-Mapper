# Product Overview: PDF Upload & Viewer

## What it does
The PDF Viewer is the starting point of the application. It provides a simple, clean drag-and-drop area where users can upload their document. Once uploaded, the document is immediately displayed on the screen, allowing users to page through it just like they would in a standard PDF reader application.

## User Benefits

* **Instant Loading:** Because the file doesn't have to travel across the internet to a server, the document appears on the screen the exact millisecond you drop it.

* **Total Privacy & Security:** The document never leaves the user's computer. It is temporarily held in the browser's memory and vanishes completely as soon as the tab is closed. This is crucial for users handling sensitive data like contracts or invoices.

* **Seamless Navigation:** Users can easily flip through multi-page documents using intuitive forward and backward controls, ensuring they can place variables exactly where they need them, no matter how long the document is.

## Business Value
By eliminating the need to store uploaded files on our servers, we completely remove the cloud hosting costs associated with file storage. It also drastically reduces legal and compliance liabilities (like GDPR) since we never take possession of the user's actual documents.
